Engagement

  1. When do you use a table?

    2. We use a table particularly when we want to show precise numerical values, managing multiple units of measure, or detailing complex information that is difficult to grasp in text format

  2. When do you not use a table?

    3. We do not use table with paragraphs, lists, or cards for textual information.

  3. When do you need to do extra work for table accessibility?

    4. We need extra work for table accessibility whenever a table moves beyond a simple grid of data (single row/column headers) or is used for non-data purposes
