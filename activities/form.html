  1. The form tag has an "action" attribute. What does it do?

    2. In HTML, the action attribute on a form tag tells the browser where to send the form data when the user submits the form.

  2. The form tag has a "method" attribute. What does it do?

    3. The method attribute on an HTML form tag tells the browser how to send the form data to the server when the form is submitted.

  3. What is the difference between a "POST" and a "GET"?

    4. GET and POST are two different HTTP request methods used when submitting a form or making a request from a browser to a server.
