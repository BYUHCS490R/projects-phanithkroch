Engagement

  1. What is the difference between block tags and inline tags?
  2. What is the margin, border, and padding?
  3. What are units of measurement you can use in CSS?
  4. When would you use a figure tag?

Answers

  1. Block tage start on a new line, take up the full width that available and stack vertically on one top of another. Inline tage, do not start on a new line, it only take up as much width as needed and stay within the same line.
  2. Margin is the space outside the element. Border is the line that wrap around the padding and content. Padding is space inside the element between the content and the border.
  3. In CSS, units of measurement are used to control size, spacing, and layout like width, height, margin, font-size, etc.
  4. We use the figure tag when we want to group media content like an image, chart, or code snippet together with a caption.
