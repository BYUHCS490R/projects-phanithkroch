Engagement - Page for Print

  1. Why is the viewpoint meta tag important?

    2. The viewport meta tag is important in modern web development because it give mobile browsers how to control the page's dimensions and scaling. Without it, websites often appear zoomed out, tiny, or improperly scaled on phones and tablets.

  2. How can you adjust the CSS for printing?

    3. To adjust the CSS for printing we can use @media print and set specific layout to hide

  3. When would you want to adjust the CSS for printing?

    4. We would want to adjust the CSS for printing whenever your webpage needs to produce a clean, readable, paper friendly version of its content.

Engagement - Responsive Pages

  1. Why do we need a viewport meta tag?

    2. We need a viewport meta tag because it: Instructs mobile devices to render the page at the correct scale Enables responsive CSS to behave properly Improves readability and usability Ensures accurate layout and touch behavior Supports modern mobile first web design

  2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

    3. When you write two CSS rules targeting the same element and both have equal specificity, the browser applies standard cascade order, meaning the rule that appears last in the stylesheet takes priority

  3. How do you change the CSS for a web page displayed on a phone?

    4. We can change the CSS for a web page on a phone by using responsive design with CSS media queries most commonly @media
