Structural Tags (HEADER, NAV, MAIN, FOOTER)

  1. What goes in the header?

    2. - It contains introductory content or a set of navigation links.

  2. What goes in the Nav? Can the Nav go in the header? Does it have to go in the header? Does the Nav contain links to another website?

    3. - Nav can be Home page, about us page, contact page, products or services, site index, or sitemap. Nav can go in the header and sometimes it does not have to go in the header. Nav is designed for internal navigation that help user move around specific website.

  3. What goes in the Main?

    4. - It can conclude page specific text and headings,

  4. -What goes in the Footer?

    5. - It can be copyright information, legal links, contact details, navigation links, author name etc.

  5. HTML Validation

    6. - HTML validation is the process of checking a web page to see if we have the right code rules defined.
