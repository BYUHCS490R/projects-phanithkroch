Internet

The Internet is a network that allows users to connect throughout the globe. It also provides global digital connectivity, allowing different devices and services to interact and share data.

Web

Web is an application that allows users to access to many websites via their internet connection. Examples include Chrome, Firefox, Safari, and Edge.

How are the Internet and Web different?

The Internet connects computers all around the world and allows users to transfer data.

Web software allows users to access, transfer, and provide their own data. The Web, which operates on the basically unnoticed Internet, is what we view and click on in our computer's browser.

ASCII and Unicode

ASCII has limitations to English, but Unicode provides global multilingual text, making it the current answer for various digital communication.

What is UTF-8

Character that supports practically all writing systems and symbols, and is the preferred internet encoding because to its efficiency and backward compatibility with ASCII.

URL

Is the unique web address for every online resource, such as a website, picture, or document, that instructs the browser how and where to discover it.

Domain Name

It is a address for a website, like google.com

HTTP

HTTP (HyperText Transfer Protocol) is the core communication language of the web

HTTPS

HTTPS is the secure version of HTTP that encrypts connections between browsers and websites, taking care of data such as passwords and credit card details.

IP Address

Every device connected to the internet is given a unique identification number known as an IP address. An IP address is a numeric code provided to internet-connected devices.

DNS

is the internet's phone book, translating human-friendly domain names (like www.google.com) into numerical IP addresses

W3C Recommendations

W3C Recommendations are official, stable web standards and guidelines developed by the World Wide Web Consortium

XML, XHTML, HTML, HTML5

XML is a coding language for storing and transmitting data that lacks predefined tags. Users may create their own tags to describe data structure, making it very flexible for data sharing between various systems such as webpages and databases.

XHTML is a rewrite of HTML as an XML application. It combines the common HTML elements with the complex syntax specifications of XML.

HTML is a standard code language for creating and structuring online content. It contains a specified set of tags that telling a web browser how to display text, pictures, and other material.

HTML5 is the current and most recent version of the HTML standard. It was designed to be more adaptable and useful than XHTML, bringing several new capabilities for modern online applications.
